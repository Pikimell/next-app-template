## Next.js базовий темплейт

Швидкий старт для нових проєктів: чистий Next.js 16 + React 19, мінімум залежностей, попередньо налаштовані ESLint і Prettier та готовий приклад сторінки.

### Що всередині
- App Router з `layout.tsx` і стартовою сторінкою.
- Глобальні стилі з приємним фоном, кнопками й картками.
- `reactStrictMode`, typed routes та відключений `x-powered-by`.
- Налаштований ESLint (core web vitals) і скрипт `format`.

### Як запустити
1. Встанови залежності: `npm install`
2. Режим розробки: `npm run dev`
3. Продакшн збірка: `npm run build` та `npm start`
4. Перевірка стилю коду: `npm run lint` або `npm run format`

### Як використовувати
- Видали або заміни вміст `app/page.tsx` на свій інтерфейс.
- Додавай нові маршрути в `app/`, компоненти в `components/` (створи папку за потреби) та публічні ресурси в `public/`.
- Онови метадані в `app/layout.tsx`, щоб вони відповідали проєкту.

### Базова структура
```
app/          # маршрути та сторінки
components/   # UI-компоненти
lib/          # утиліти та інфраструктурний код
hooks/        # спільні React хуки
store/        # стан (zustand/redux/інше)
services/     # API-клієнти та запити
types/        # спільні типи/інтерфейси
utils/        # дрібні допоміжні функції
public/       # статичні файли
```
